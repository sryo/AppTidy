# AppTidy

A macOS status bar app that keeps your workspace clean.

## Features

**AppTimeout**  
Automatically quits apps that have no open windows after a set idle time. Includes a whitelist and smart audio protection so your music never stops.

**UndoClose**  
Accidentally quit an app or closed a Finder window? Press **⌘⌥Z** within 5 seconds to bring it back.
